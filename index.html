Privacy Policy for Vegan Groups Scraper

Last Updated: 04.04.2025

1. Overview

This Privacy Policy explains how the Vegan Groups Scraper collects, uses, and protects data when accessing public Facebook group data for academic research purposes.

2. Data Collection and Use

Anonymized Data: We do not collect or store real user names or other personal identifiers. Instead, each user is assigned an anonymized ID.

Consistent IDs: The anonymized ID for each user is consistent over time, allowing us to track their posts and comments across multiple sessions. This is done solely for the purpose of analyzing discourse trends in a non-identifiable manner.

Data Storage: All collected data is stored securely and is used only for academic research. We do not share or sell this data to third parties.

3. Data Protection

Anonymization: No personal identifiers are saved. All user data is anonymized before analysis.

Security: We employ appropriate technical and organizational measures to safeguard the data against unauthorized access.

4. User Rights

Since the data collected is anonymized and used solely for research purposes, it does not include personally identifiable information. Nonetheless, we adhere to all applicable data protection laws and ethical guidelines.

5. Contact Information

If you have any questions regarding this Privacy Policy or the data practices of the Vegan Groups Scraper, please contact us at: kata@hey.com.
